### Basic Commands:
To generate / show execution plan
```
terraform plan
```
---
To execute a terraform
```
terraform apply
```
---
To execute a terraform script without confirmation
```
terraform apply --auto-approve
```
---
To destroy all the configs
```
terraform destroy
```
